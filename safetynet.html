@safetynet — project overview

@safetynet is a project that aims to promote and facilitate the @deso blockchain.

Officially first ever DeSo registered Foundation at Dutch Chamber of Commerce under number: 97663069.

The @safetynet project creates a system of redundancy for the @deso ecosystem with a secure decentralized network owned and operated by the community. The @safetynet project is a project started by members of @thedc.

Our main objectives

  1. Securing the blockchain by providing redundancy in all key components of the @deso blockchain.
  2. Making sure all user content will stay safe at any time and regardless of any unfortunate incident occurring.
  3. Decentralizing the @deso blockchain by providing extra servers and all related services so we all are not depending on one single or few point(s) of failure.
  4. Building the @deso ecosystem by collaborations with and support to all members of the @deso community (individuals, node operators and developers).
  5. Testing the resilience of the @deso blockchain.
  6. Adding functionality to the blockchain by providing on chain software solutions.
